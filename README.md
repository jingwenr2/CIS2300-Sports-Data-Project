# CIS2300 Sports Data Analysis Project

## Overview
This Python project processes sports data using pandas, matplotlib, and Wikipedia API integration.

## Features
- Load and clean CSV sports dataset
- Filter by selected countries and sports
- Generate bar chart visualizations
- Add and delete records dynamically
- Save updated data
- Retrieve Wikipedia summaries via API

## Technologies Used
- Python
- Pandas
- Matplotlib
- Wikipedia API

## How to Run
1. Install required packages:
   pip install pandas matplotlib wikipedia-api

2. Run the program:
   python Ruan_Jingwen_program.py
